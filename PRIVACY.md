# Privacy Policy for FinTrack

**Last Updated:** March 10, 2026

FinTrack (“the App”) is a personal finance tracking application designed to help users manage income, expenses, and loans. This Privacy Policy explains how information is collected, used, and protected when using the application.

By using FinTrack, you agree to the collection and use of information in accordance with this Privacy Policy.

---

# Information Collection

FinTrack collects only the information necessary to provide its core functionality.

### Account Information
When creating an account, users authenticate using email or Apple Sign-In. Authentication and account management are handled through Supabase. The following information may be collected and associated with the user's account:

- Email address
- Authentication identifier generated by Supabase
- Apple Sign-In identifier (if Apple authentication is used)

This information is linked to the user's identity in order to enable secure login, data synchronization across devices, and account management.

### Financial Data
Users may store personal financial information in the app, including:

- Income records
- Expense records
- Loan records (borrowed or lent amounts)
- Categories and notes related to financial entries

This information is created and managed entirely by the user and is stored in the user's secure account database.

### Device Security Data
FinTrack may use device-level security features such as:

- Face ID authentication
- Local PIN codes

These features are processed locally on the user's device and are not transmitted to external servers.

---

# Data Usage

The information collected is used strictly for the purpose of operating and improving the FinTrack application.

Specifically, the data is used to:

- Authenticate users and manage accounts
- Synchronize financial records across devices using Supabase
- Provide core application features such as income, expense, and loan tracking
- Maintain application security and prevent unauthorized access
- Enable users to access their personal financial records from multiple devices

FinTrack does **not sell, rent, or trade user data** to third parties.

User financial data is only accessible to the authenticated user associated with the account.

---

# Data Security

FinTrack takes reasonable measures to protect user information.

Security measures include:

- Secure authentication provided through Supabase
- Encrypted communication between the application and cloud services
- Local device security using Face ID and optional PIN protection
- Access controls ensuring users can only access their own data

Despite these protections, no method of internet transmission or electronic storage is completely secure. Therefore, absolute security cannot be guaranteed.

---

# User Rights

Users retain full control over their data stored in FinTrack.

Users have the right to:

- Access their stored financial data within the application
- Modify or delete individual financial records at any time
- Delete their account and all associated data directly from the app settings

When an account is deleted, all user data associated with the account is permanently removed from the system.

---

# Third-Party Services

FinTrack relies on the following third-party service:

**Supabase**

Supabase provides authentication and secure cloud data storage. When using the app, certain account identifiers (such as email or Apple authentication identifiers) are processed through Supabase to enable login and synchronization functionality.

Users are encouraged to review Supabase's privacy policies for more information on how that service manages data.

---

# Children's Privacy

FinTrack is not directed toward children under the age of 13. The app does not knowingly collect personal information from children.

If it becomes known that personal information from a child under 13 has been collected without parental consent, appropriate steps will be taken to remove that information.
